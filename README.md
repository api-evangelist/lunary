# Lunary (lunary)

Lunary is an open-source LLM observability and evaluation platform: tracing, prompt management, datasets, evaluations, and analytics for LLM apps. Available as a hosted SaaS at lunary.ai and as a self-hosted Apache 2.0 deployment. Exposes a REST API authenticated with Bearer tokens.

**APIs.json:** [https://raw.githubusercontent.com/api-evangelist/lunary/refs/heads/main/apis.yml](https://raw.githubusercontent.com/api-evangelist/lunary/refs/heads/main/apis.yml)

## APIs

### Lunary API

REST API for Lunary covering ingestion (logs/traces), prompts (template management with versions and labels), datasets, evaluations, projects, users, and analytics. Authentication uses Bearer tokens; private API keys for management endpoints, public keys for ingestion.

- **Human URL:** [https://docs.lunary.ai/api](https://docs.lunary.ai/api)
- **Base URL:** `https://api.lunary.ai/v1`
